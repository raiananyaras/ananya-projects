# 📚 Study Buddy — AI Doubt Solver (Frontend Demo)

## 🚀 Project Overview

Study Buddy is a frontend-only web application that helps students solve doubts using AI. It supports both text-based and image-based questions by integrating with the Gemini API (REST).

> ⚠️ **Security Note:**
> This demo directly calls the Gemini API from the browser. For production use, API requests should be handled via a backend to protect the API key.

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Gemini API (REST)
* OCR (for image-based questions)

---

## ✨ Features

* 💬 Ask questions using text input
* 🖼️ Upload images to extract and solve doubts
* 🤖 AI-powered responses using Gemini API
* ⚡ Fast and responsive UI
* 📱 Works on desktop and mobile
